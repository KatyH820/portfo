002 :Columns Game

In the late 1980s, a game called Columns appeared on at least one personal computing platform. It was then ported to the Sega Genesis and later gained a bit of popularity when it was ported to other video game consoles and PCs, becoming one of the more famous puzzle games of the early 1990s. In this program, I coded a version of the game that is playable and with a user interface in Python. I separate the logic and the display into different files. I create two classes in my logic files to keep track of and modify the faller and game state. Finally, I make my user interface with Pygame and give a kind of visual cue when fallers land and freeze.

To run this program, you need to install Pygame, or the animation won't work. Feel free to download the code of my program on my GitHub.
